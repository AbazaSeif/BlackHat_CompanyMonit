Read First

The ThreadLifeCycle.zip file contains the following files:

  1. CompileRun_ MultiThreaded.bat - a batch file that when double clicked will compile and display the MultiThreaded application.
  2. CompileRun_ MultiThreaded_II.bat - a batch file that when double clicked will compile and display the MultiThreaded_II application.
  3. CompileRun_ MultiThreaded_III.bat - a batch file that when double clicked will compile and display the MultiThreaded_II application.
  4. CompileRun_ MultiThreaded_IV.bat - a batch file that when double clicked will compile and display the compiled MultiThreaded_IV application.
  5. CompileView_ MultiThreaded -
    • The compiled file is the MultiThreaded class file.
  6. ThreadLifeCycle_Help.html - this file provides help on how to get started.
  7. MultiThreaded.java - the Java source file for the MultiThreaded.java.java program.
  8. MultiThreaded_II.java - the Java source file for the MultiThreaded_II.java.java program.
  9. MultiThreaded_III.java - the Java source file for the MultiThreaded_III.java.java program.
  10. MultiThreaded_IV.java - the Java source file for the MultiThreaded_IV.java.java program.
  11. MultiThreaded_Tutorial.html - a file that describes how to build the ThreadLifeCycle.java program.
  12. run_Javadoc_MultiThreaded.bat - the batch file that executes the Javadoc file for the MultiThreaded.java file.
  13. run_Javadoc_MultiThreaded_II.bat - the batch file that executes the Javadoc file for the MultiThreaded_II.java file.
  14. run_Javadoc_MultiThreaded_III.bat - the batch file that executes the Javadoc file for the MultiThreaded_III.java file.
  15. run_Javadoc_MultiThreaded_IV.bat - the batch file that executes the Javadoc file for the MultiThreaded_IV.java file.

How to compile and run

There are two ways to run the programs found in this .zip file.

  1. HelloWorld program:
    1. You can double-click on the CompileView_HelloWorld.bat file that is supplied with the ThreadLifeCycle.zip file.
    2. You can load the HelloWorld.java and HelloWorld_I.html files into an IDE and run the program from that environment.
  2. ThreadLifeCycle program:
    1. You can double-click on the CompileView_ThreadLifeCycle.bat file that is supplied with the ThreadLifeCycle.zip file.
    2. You can load the ThreadLifeCycle.java and THREADLC.html files into an IDE and run the program from that environment.
  3. ThreadLifeCycle_II program:
    1. You can double-click on the CompileView_ThreadLifeCycle_II.bat file that is supplied with the ThreadLifeCycle.zip file.
    2. You can load the ThreadLifeCycle_II.java and THREADLC_II.html files into an IDE and run the program from that environment.
  4. ThreadLifeCycle_III program:
    1. You can double-click on the CompileView_ThreadLifeCycle_III.bat file that is supplied with the ThreadLifeCycle.zip file.
    2. You can load the ThreadLifeCycle_III.java and THREADLC_III.html files into an IDE and run the program from that environment.

If you do not have the Java documentation

The Help and Tutorial is not intended to repeat the documentation found in:

  1. J2SE 1.3.1
  2. JavaTM 2 Platform, Standard Edition, v 1.4.2 API Specification
  3. JDKTM 5.0 Documentation

How to Obtain Java

If you do not have the Java installed, you can download a copy from:

  1. J2SE 1.3.1
  2. Java(TM) 2 SDK, Standard Edition 1.4.2_08
  3. Java SE 5.0 Downloads

How to Obtain a Free Java IDE

If you do not have the Java IDE installed, you can download a copy from:

  1. Sun's Award-Winning Developer Tools
  2. Borland® JBuilder®
